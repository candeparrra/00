1- ¿Cuáles son los tipos de títulos que hay?

Los titulos son 6: h1, h2, h3, h4, h5 y h6.

2- ¿Cuál es la importancia del número que lleva la etiqueta de título?

La importancia es que el numero determina su prioridad, siendo h1 el mas importante y h6 el menos importante.

3- ¿Qué es lo que debe evitarse para que el uso de los títulos sea accesible a las personas no videntes?. Justificar

Para las personas nos videntes es mucho mas accesible leer (a traves de un dispositivo) si los titulos estan ordenados de mayor a importancia, a menor. Por eso se recomienda evitar saltearse titulos.
Ejemplo de lo que no hay que hacer:
h1 "texto 1"
h4 "texto 2"

4- ¿Cuál es el concepto de anidar títulos? Ejemplificar con un ejemplo distinto al propuesto por MDN.

Anidar titulos es realizar sub secciones con el objetivo de mejorar la comprensión de organizacion o jerarquia para los lectores. Ejemplo:

5- ¿Qué es un atributo en una etiqueta HTML? Dar dos ejemplos de etiquetas HTML con atributos, marcando sus respectivos atributos.

Un atributo otorga informacion adicional, caracteristicas especificas o condiciones a una etiqueta. Ejemplos:

html lang="en"

Siendo "lang", un atributo, y "en", su valor.

meta name="hola"

Siendo "name" un atributo, y "hola", su valor.

6- ¿Qué atributos tienen las etiquetas de título?. Definir.

Las etiquetas de titulo tienen atributos globales, quiere decir que no solo pueden aplicarse en titulos sino que tambien en otras etiquetas HTML. Ejemplos: class, permite que CCS y JavaScript accedan a elementos específicos a través de los selectores de clase o funciones. Otro ejemplo es id, este "identificador" debe ser unico en todo el documento, se usa para identificar un elemento al vincularlo, al escribir un script o aplicarle estilo.

7- ¿Qué hace la etiqueta "p"? ¿Cuáles son sus atributos?

La etiqueta P distribuye el texto en parrafos. Sus atributos son; Title, style, id, class, dir y lang.

8- Investiga qué hacen y cómo se utilizan las siguientes etiquetas HTML: strong i u sub sup em a span

strong: es aplicado para generar enfasis en las partes mas importantes de un texto.

i: cambia la letra del texto a estilo cursiva o italica.

u: subraya el texto.

sub: muestra el fragmento de texto que quieras en un tamaño mas pequeño y mas bajo, un subindice.

sup: muestra el fragmento de texto que quieras en un tamaño mas pequeño y mas alto, un superindice.

em: le agrega enfasis a las partes importantes de un texto cambiando su tipografia.

a: crea enlaces de paginas webs, archivos, ubicaciones u otras URL.

span: le agrga estilo a un texto o palabras, por ejemplo, cambia su color.

2- ¿Qué es el HTML? ¿En qué consiste?

HTML es un lenguaje de enmarcado o una estructura para crear paginas web, consta de elementos y etiquetas que determinan muchas cosas, como la manera en la que aparecera el texto, el tamaño, o el color. Pueden agregar enlaces, y encerrar o delimitar las partes del contenido que desees, entre otras funciones.

3- ¿Qué es una etiqueta? ¿Funciona si se escribe tanto en mayúscula como en minúscula? ¿Cuál es la mejor práctica para escribir etiquetas?

Una etiqueta sirve para determinar la manera en la que se mostrará el contenido de la pagina que quisieramos crear, funciona si se escribe en mayusucula o en minuscula pero por conveniencia se escriben en minuscula, se escriben dentro de simbolos mayor y menor, y siempre debe tener su etiqueta de cierre (la cual tendrá una barra /), la primera etiqueta que abrimos, será la primera que cerremos. Las etiquetas deben estar anidadas, el contenido de la misma debe tener un espacio antes, asi identificaremos que está dentro de la etiqueta. Algunas etiquetas deben tener si o si un atributo.

4- ¿A qué llamamos elemento HTML? ¿Cuál es su anatomía? Explicar sus partes.

Un elemento HTML es un elemento de una pagina web, este puede contener datos, textos o una imagen. Su anatomía consta de una etiqueta de apertura, contenido y una etiqueta de cierre. Las etiquetas sirven para determinar la manera en la que queremos que se muestre el contenido de la pagina y dentro de estas, el contenido deseado.

11- ¿Para qué sirve la etiqueta img? ¿Cuáles son sus atributos? ¿Para qué sirven cada uno de ellos?

La etiqueta img sirve para agregar imagenes, tiene varios atributos pero requiere por lo menos uno para utilizarla, los mas conocidos son: SCR, Source es el mas importante, este muestra la imagen ya que contiene la ruta que la dirige. ALT, su valor debe ser la descripcion textual de la imagen, sirve para cuando una imagen tarda mucho en ser mostrada, para los usuarios con discapacidad visual, o cuando la pagina no acepta el tipo de imagen, entre otros. La etiqueta img solo tiene etiqueta de apertura.
